Unlimited Calendars

You take on different roles in different settings. Create a calendar for each role or function. Control the visibility of each.

Unlimited Events

There are 365 days in a year, and countless ways to divide it. Create unlimited events across unlimited calendars.

Color Coding

Use a color selected to represent a specific calendar, or use a predefined color representing the month. Switch with a click.

Event List

View events in a single, vertical list alongside the year view. Selecting an event from one view, maps to the other view.

Search Events

Search event names and notes for any string. Matches are filtered in the event list and the year view. It is awesome!

Detail Oriented

More robust than a spreadsheet, you can add details for location, link, attachments, and more to go along with your events.

Privacy

All your data is kept local on your device unless you specifically enable event sharing or sync. Backup/sync data is encrypted.

Mobile and Desktop

Anno Awesome is designed to work with the browser on desktop and tablet devices.

Secure and Flexible

Work securely in any modern browser. Install on supported devices for that home screen icon and full screen display.
